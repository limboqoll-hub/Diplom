# Viveski

## Требования

- Node.js (рекомендуется версия 18 или выше)
- npm

## Установка зависимостей

```
npm install
```

## Запуск

### Режим разработки

```
npm run dev
```

Открыть в браузере: http://localhost:5173

### Сборка для продакшена

```
npm run build
```

### Предпросмотр сборки

```
npm run preview
```
